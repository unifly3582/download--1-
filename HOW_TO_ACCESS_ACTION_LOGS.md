## Troubleshooting

**Don't see "Action Logs" in sidebar?**
- Refresh your browser (Ctrl+R or Cmd+R)
- Clear browser cache
- Restart your development server

**Page not loading?**
- Make sure you're logged in as admin
- Check browser console for errors
- Verify the URL is correct
